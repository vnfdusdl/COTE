# COTE

### 소개 
- 게터디 팀원들과 함께하는 스터디입니다. 
- 참가자 전원 코드 발표를 진행하며 어려웠던 부분, 새로 알게 된 부분을 공유합니다.
- 에러, 오작동, 메모리 효율, 코드의 재사용, 올바른 자바스크립트 문법 등을 고려하며 리뷰를 진행합니다. 

### 목적
- Node.js로 백준을 풀면서 자바스크립트를 익히고 프로그래밍 사고를 키웁니다. 
- 점차 단계를 올려서 코딩테스트도 대비합니다.
